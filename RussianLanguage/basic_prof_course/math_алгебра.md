## 

**Элемент множество**  
которые составляют множество, называются элементами множества.

записывают: $B \subset A$  
читают: x принадлежит множеству A, или, x - элемент множества A.  
предмет x **принадлежит** множеству A

записывают: $B \not \subset A$
читают: x не принадлежит множеству A, или, x не элемент множества A.  
предмет x **не принадлежит** множеству A

### Основные определения

синус / синусом угла (sin): $\sin \alpha = \dfrac{противолежащий \ катет}{гипотенуза}$
косинус / косинусом угла (cos): $\cos \beta = \dfrac{прилежащий \ катет}{гипотенуза}$
тангенс / тангенсом угла (tg\tan): $\tan x = \dfrac{противолежащий \ катет}{прилежащий \ катет} = \dfrac{\sin x}{\cos x}$
котангенс / котангенсом угла (ctg\cot): $\cot x = \dfrac{прилежащтий \ катет}{противолежащий \ катет} = \dfrac{\cot x}{\sin x}$
секанс: $ \sec \alpha $
косеканс: $ \cosec \alpha $
